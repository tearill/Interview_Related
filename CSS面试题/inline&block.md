# 行内元素和块级元素  

## 行内元素  
- 和别的元素在同一行  
- 不可以设置宽高，宽高由内容决定，可以设置行高  
- margin 和 bottom 在水平方向设置有效，在垂直方向设置是视觉上有效，但是不会影响其他元素的布局(实际无效)  
- 行内元素不能包含块级元素，只能包含文本或者其他行内元素  
- 常见行内元素：input、span、a、em、i  

## 块级元素  
- 块级元素单独占据一行，宽度默认 100%，高度由内容决定，垂直向下排列  
- 可以设置宽高  
- 可以包含行内元素和块级元素  
- 常见的块级元素：div、p、ul、li、h[1-6]  

## 行内块级元素  
同时具备内联元素、块状元素的特点
